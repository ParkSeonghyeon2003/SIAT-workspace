인스턴스 변수와 클래스 변수

이 교안은 인스턴스 변수와 클래스 변수의 차이, 문법, 예제, 퀴즈/과제를 한 번에 정리합니다.

1) 개념

인스턴스 변수
  • 객체(인스턴스)마다 독립적으로 존재하는 변수입니다.
  • self.변수명 형태로 정의됩니다.
  • 같은 클래스로 만든 인스턴스라도 값이 달라질 수 있습니다.
예: 학생 1의 name과 학생 2의 name은 서로 다를 수 있습니다.
클래스 변수
  • 클래스 전체가 공유하는 변수입니다.
  • 클래스명.변수명 또는 클래스 내부에 정의합니다.
  • 모든 인스턴스가 같은 값을 봅니다.
예: 같은 학교 소속이라면 school_name은 모두 동일하게 공유됩니다.

한 줄 요약

인스턴스 변수는 “각자(개별 데이터)”, 클래스 변수는 “공통(공유 데이터)”입니다.

2) 문법

기본 형태

클래스 변수는 클래스 블록에 작성하고, 인스턴스 변수는 보통 __init__에서 self로 작성합니다. 

class 클래스명:
    클래스변수 = 값  # 클래스 변수

    def __init__(self, 값):
        self.인스턴스변수 = 값  # 인스턴스 변수
기억
- self.가 붙으면 보통 “인스턴스 변수”입니다.
- 클래스 변수는 클래스명.변수명으로 접근/변경합니다.

3) 예시

Student 예제

school_name은 모두가 공유(클래스 변수), name/age는 각자 보유(인스턴스 변수)입니다. 

class Student:
    school_name = "파이썬고등학교"   # 클래스 변수

    def __init__(self, name, age):
        self.name = name           # 인스턴스 변수
        self.age = age             # 인스턴스 변수

# 인스턴스 생성
s1 = Student("홍길동", 18)
s2 = Student("김철수", 17)

# 인스턴스 변수 출력 (개별적)
print(s1.name, s1.age)   # 홍길동 18
print(s2.name, s2.age)   # 김철수 17

# 클래스 변수 출력 (공유됨)
print(s1.school_name)    # 파이썬고등학교
print(s2.school_name)    # 파이썬고등학교

# 클래스 변수 변경
Student.school_name = "AI고등학교"

print(s1.school_name)    # AI고등학교
print(s2.school_name)    # AI고등학교
포인트
클래스 변수를 바꾸면(Student.school_name = ...) 만들어진 인스턴스들도 같은 값을 봅니다.
