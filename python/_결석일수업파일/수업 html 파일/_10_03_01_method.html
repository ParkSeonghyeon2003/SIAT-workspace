파이썬 메서드 종류 Instance / Class / Static

클래스에서 자주 쓰는 3가지 메서드(인스턴스, 클래스, 정적 메서드)를 개념 → 문법 → 예시 순서로 정리합니다.

1. 개념

파이썬 클래스에서 메서드는 크게 세 가지 종류로 나눌 수 있습니다.

  1. 인스턴스 메서드 (Instance Method)
    • 가장 일반적인 메서드입니다.
    • 첫 번째 매개변수로 self를 받으며, 인스턴스 변수에 접근 가능합니다.
    • 특정 객체(인스턴스)와 연관되어 동작합니다.
  2. 클래스 메서드 (Class Method)
    • 클래스 자체와 관련된 동작을 정의할 때 사용합니다.
    • @classmethod 데코레이터를 사용합니다.
    • 첫 번째 매개변수로 cls를 받아 클래스 변수에 접근 가능합니다.
  3. 정적 메서드 (Static Method)
    • 클래스/인스턴스에 직접적으로 의존하지 않는 동작을 정의합니다.
    • @staticmethod 데코레이터를 사용합니다.
    • self나 cls를 받지 않으며, 일반 함수와 비슷하지만 클래스 안에 포함됩니다.
핵심 요약:
인스턴스 메서드는 self로 객체 상태를 다루고, 클래스 메서드는 cls로 클래스 공통 데이터를 다루며, 정적 메서드는 클래스/객체 상태와 무관한 기능을 담을 때 사용됩니다.

2. 문법

class MyClass:
    # 인스턴스 메서드
    def instance_method(self):
        print("인스턴스 메서드 호출")

    # 클래스 메서드
    @classmethod
    def class_method(cls):
        print("클래스 메서드 호출")

    # 정적 메서드
    @staticmethod
    def static_method():
        print("정적 메서드 호출")

3. 메서드 종류 한눈에 보기

구분 데코레이터 첫 번째 매개변수 접근 가능 대상 주 사용 목적
인스턴스 메서드 없음 self 인스턴스 변수 객체(인스턴스)의 상태를 처리합니다.
클래스 메서드 @classmethod cls 클래스 변수 클래스 공통 정보/규칙을 처리합니다.
정적 메서드 @staticmethod 없음 없음(직접 접근 불가) 독립적인 기능(유틸 함수)을 제공합니다.

4. 예시

① 인스턴스 메서드 (Instance Method)

  • 첫 번째 매개변수로 self를 받습니다.
  • self.name, self.age처럼 인스턴스 변수에 접근 가능합니다.
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def introduce(self):
        print(f"안녕하세요, 저는 {self.name}이고, 나이는 {self.age}살입니다.")

p1 = Person("지민", 25)
p1.introduce()

출력

안녕하세요, 저는 지민이고, 나이는 25살입니다.
포인트:
인스턴스 메서드는 “이 객체의 name, age”처럼 객체별로 다른 값을 다룰 때 사용됩니다.

② 클래스 메서드 (Class Method)

  • @classmethod를 붙입니다.
  • 첫 번째 매개변수는 cls이며, 클래스 변수에 접근 가능합니다.
  • 인스턴스 변수(self.name 등)에는 직접 접근할 수 없습니다.
class Person:
    species = "인간"

    def __init__(self, name, age):
        self.name = name
        self.age = age

    @classmethod
    def show_species(cls):
        print(f"이 클래스의 종은 {cls.species}입니다.")

Person.show_species()  # 클래스에서 직접 호출

p1 = Person("지민", 25)
p1.show_species()      # 인스턴스에서도 호출 가능

출력

이 클래스의 종은 인간입니다.
이 클래스의 종은 인간입니다.
주의:
클래스 메서드는 cls로 클래스 변수에 접근합니다. 따라서 객체마다 달라지는 인스턴스 변수는 다루기 어렵습니다.

③ 정적 메서드 (Static Method)

  • @staticmethod를 붙입니다.
  • self, cls를 받지 않습니다.
  • 클래스/인스턴스 변수에 의존하지 않는 기능을 클래스 안에 넣고 싶을 때 사용됩니다.
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    @staticmethod
    def is_adult(age):
        return age >= 18

print(Person.is_adult(20))  # True

p1 = Person("지민", 25)
print(p1.is_adult(15))      # False
설명:
정적 메서드는 “나이(age)를 받아 성인인지 판단하기”처럼, 클래스나 객체 상태와 관계없이 동작하는 기능을 담을 때 유용합니다.

복합 예제 (Instance + Class + Static)

class Student:
    school_name = "파이썬 고등학교"  # 클래스 변수

    def __init__(self, name, age):
        self.name = name      # 인스턴스 변수
        self.age = age

    # 인스턴스 메서드
    def introduce(self):
        print(f"안녕하세요, 제 이름은 {self.name}이고 {self.age}살입니다.")

    # 클래스 메서드
    @classmethod
    def school_info(cls):
        print(f"저희 학교는 {cls.school_name}입니다.")

    # 정적 메서드
    @staticmethod
    def welcome():
        print("학생 여러분, 환영합니다!")

# 사용 예시
s1 = Student("홍길동", 18)
s1.introduce()        # 인스턴스 메서드 호출

Student.school_info() # 클래스 메서드 호출
Student.welcome()     # 정적 메서드 호출
정리:
  • introduce()는 학생 개인(인스턴스)의 이름/나이를 사용합니다.
  • school_info()는 학교 이름처럼 클래스 공통 정보를 사용합니다.
  • welcome()은 상태와 무관한 안내 문구를 출력합니다.

5. 실전 선택 기준

어떤 메서드를 선택하면 되나요?

  • 객체(인스턴스)마다 값이 다르다 → 인스턴스 메서드를 사용합니다.
  • 클래스 공통 규칙/설정 → 클래스 메서드를 사용합니다.
  • 상태와 무관한 유틸 기능 → 정적 메서드를 사용합니다.
추가 팁:
“클래스 안에 넣긴 했는데 self/cls가 필요 없다”면 정적 메서드로 두는 편이 의도가 더 명확해집니다.

자주 나오는 질문

Q. 클래스 메서드는 인스턴스에서도 호출 가능한가요?
A. 호출은 가능하지만, 내부에서는 cls로 클래스 데이터만 다룹니다.
Q. 정적 메서드는 왜 클래스 안에 넣나요?
A. 기능이 해당 클래스와 “논리적으로 관련”이 있을 때, 묶어두면 구조가 깔끔해집니다.
