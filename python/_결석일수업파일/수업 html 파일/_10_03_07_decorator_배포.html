파이썬 데코레이터(Decorator)

데코레이터의 개념 → 문법 → 예시 순서로 정리합니다.

1. 개념

  • 정의: 함수를 인자로 받아 새로운 함수를 반환하는 함수입니다.
  • 역할: 원래 함수를 수정하지 않고 기능을 확장할 수 있습니다.
  • 활용 예시: 로깅(logging), 권한 체크, 실행 시간 측정 등입니다.
  • 동작 흐름: 원래 함수 → 데코레이터 → 새로운 함수
핵심:
데코레이터는 “기존 함수 코드를 손대지 않고”, 함수 실행 전/후에 동작을 끼워 넣는 방식으로 확장합니다.

2. 문법

def 데코레이터이름(func):
    def wrapper(*args, **kwargs):
        # 실행 전 처리
        print("함수 실행 전 동작")

        result = func(*args, **kwargs)

        # 실행 후 처리
        print("함수 실행 후 동작")

        return result
    return wrapper

@데코레이터이름
def 함수이름():
    print("원래 함수 실행")

함수이름()
@ 문법은 아래 코드와 동일하게 동작합니다.
함수이름 = 데코레이터이름(함수이름)
이름 의미
func 원래 함수를 가리킵니다.
wrapper 원래 함수를 감싸는 함수(새로 반환되는 함수)입니다.
*args, **kwargs 원래 함수의 인자를 그대로 전달하기 위해 사용합니다.
주의:
*args, **kwargs를 사용하면 “인자가 있는 함수”도 동일한 데코레이터로 처리할 수 있습니다.

3. 예시

(1) 지금까지 사용하던 호출 방식

def hello_decorator(func):
    def wrapper():
        print("함수 시작!")
        func()
        print("함수 끝!")
    return wrapper

def say_hello():
    print("안녕하세요!")

say_hello = hello_decorator(say_hello)
say_hello()

출력

함수 시작!
안녕하세요!
함수 끝!
설명:
say_hello = hello_decorator(say_hello)로 say_hello가 wrapper로 교체됩니다.

(2) @ 문법으로 간단히 호출

def hello_decorator(func):
    def wrapper():
        print("함수 시작!")
        func()
        print("함수 끝!")
    return wrapper

@hello_decorator
def say_hello():
    print("안녕하세요!")

say_hello()
포인트:
@hello_decorator는 say_hello = hello_decorator(say_hello)와 동일하게 동작합니다.

(3) 인자를 받는 함수 (*args 사용 예제)

def decorator_with_args(func):
    def wrapper(*args, **kwargs):
        print("함수 시작!")
        result = func(*args, **kwargs)
        print("함수 끝!")
        return result
    return wrapper

@decorator_with_args
def greet(name):
    print(f"{name}님, 안녕하세요!")

greet("지민")

출력

함수 시작!
지민님, 안녕하세요!
함수 끝!
정리:
*args, **kwargs 덕분에 인자 있는 함수도 문제없이 데코레이션 가능합니다.

(4) 인자를 받는 함수 (*args, **kwargs 사용 예제)

def greet_decorator(func):
    def wrapper(*args, **kwargs):
        print("함수 시작!")

        # kwargs에서 특정 키를 확인하고 사용
        if 'greeting' in kwargs:
            print(f"특별 인사: {kwargs['greeting']}")

        # 원래 함수 실행
        result = func(*args, **kwargs)

        print("함수 끝!")
        return result
    return wrapper

@greet_decorator
def greet(name, **kwargs):
    print(f"{name}님, 안녕하세요!")

# 호출 예제 1: 일반 호출
greet("지민")
print("---")

# 호출 예제 2: 키워드 인자 포함
greet("민수", greeting="오늘 하루도 화이팅!")

출력

함수 시작!
지민님, 안녕하세요!
함수 끝!
---
함수 시작!
특별 인사: 오늘 하루도 화이팅!
민수님, 안녕하세요!
함수 끝!

(5) 실행 시간 측정

import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"실행 시간: {end - start:.4f}초")
        return result
    return wrapper

@timer
def slow_function():
    time.sleep(2)
    print("작업 완료!")

slow_function()

출력 예

작업 완료!
실행 시간: 2.0003초
참고:
실행 시간은 환경에 따라 소수점 값이 달라질 수 있습니다.
