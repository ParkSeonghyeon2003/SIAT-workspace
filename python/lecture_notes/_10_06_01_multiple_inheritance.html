🧬 파이썬 다중 상속 (Multiple Inheritance)

1. 코드 핵심 설명

제공된 코드는 자식 클래스 C가 두 개의 부모 클래스 A와 B를 동시에 상속받는 전형적인 다중 상속 구조입니다.

구현 포인트: class C(A, B): 문법을 통해 부모 클래스들을 쉼표로 구분하여 나열합니다. 이로 인해 클래스 C의 인스턴스는 부모들의 모든 자원을 자신의 것처럼 사용할 수 있게 됩니다.

2. 멤버 접근 권한 (Method Availability)

클래스 사용 가능한 메서드 비고
Class A method_a() 독립적인 부모 클래스
Class B method_b() 독립적인 부모 클래스
Class C method_a(), method_b(), method_c() A와 B의 기능을 통합

3. 다중 상속 시 주의사항 (Diamond Problem)

다중 상속은 강력하지만, 만약 A와 B에 이름이 같은 메서드가 있다면 어떤 것을 먼저 호출할지 결정하는 MRO(Method Resolution Order, 메서드 결정 순서) 문제가 발생합니다.

  • 파이썬은 C.mro()를 통해 탐색 순서를 결정합니다.
  • 일반적으로 (A, B) 순서로 적었다면 A를 먼저 탐색합니다.
