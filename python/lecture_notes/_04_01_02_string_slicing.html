Python 문자열 슬라이싱(Slicing)

문자열의 일부 구간을 잘라내는 방법

1. 슬라이싱이란?

👉
슬라이싱(Slicing)이란 문자열(또는 리스트)의 일부 구간을 잘라내어 추출하는 기능입니다.

기본 형식

문자열[시작 : 끝 : 간격]
시작(start) 슬라이싱을 시작할 인덱스 (포함)
끝(end) 슬라이싱을 끝낼 인덱스 (❗끝-1 까지 포함)
간격(step) 몇 칸씩 이동할지 결정 (기본값 = 1)
✅ 끝 인덱스는 포함되지 않는다 → start ~ end-1

2. 기본 슬라이싱 예제

text = "Python Programming"

print(text[0:6])   # Python
print(text[:6])    # Python
print(text[7:])    # Programming
text = "Python Programming" index: 0 1 2 3 4 5 6 7 ... 결과: text[0:6] → Python text[7:] → Programming

3. 간격(step) 활용

print(text[::2])   # Pto rgamn
print(text[::-1])  # gnimmargorP nohtyP
[::2] 처음부터 끝까지 2칸씩 건너뛰며 출력
[::-1] 문자열을 역순(뒤집기) 출력
✅ 간격이 음수이면 뒤에서 앞으로 역방향 이동합니다.

4. 음수 인덱스 활용

print(text[1:-1])  # ython Programmin
음수 인덱스: -1 → 마지막 문자 -2 → 뒤에서 두 번째 문자
✅ [1:-1] → 첫 글자와 마지막 글자를 제외한 전체 출력

5. 숫자 문자열 예제

text = "0123456789abcde"

print(text[0:3])    # 012
print(text[:3])     # 012
print(text[3:4])    # 3
print(text[:3:2])   # 02
print(text[::2])    # 02468ace
print(text[:-1])    # 0123456789abcd
print(text[-1])     # e
[0:3]0~2까지 출력
[:3:2]처음부터 2칸씩 → 02
[:-1]마지막 문자 제외
[-1]마지막 문자만 출력

6. 정리

[start:end] start부터 end-1까지
[:end] 처음부터 end-1까지
[start:] start부터 끝까지
[::step] 간격(step)만큼 건너뛰기
[::-1] 문자열 뒤집기
✅ 슬라이싱은 문자열뿐 아니라 리스트, 튜플에도 동일하게 사용됩니다.
