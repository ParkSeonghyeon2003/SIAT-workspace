Python 리스트(list) 교안

개념 · 생성 · 출력 · 인덱스 · 변경(추가/수정/삭제) · 리스트 vs 튜플

list mutable index append remove

1) 리스트(list)란?

  • 여러 값을 순서대로 저장할 수 있는 자료형입니다.
  • [ ] (대괄호)로 만들고, 값은 쉼표로 구분합니다.
  • 인덱스(index)가 있으며, 0부터 시작합니다.
  • mutable(변경 가능) → 추가/수정/삭제가 가능합니다.
  • 서로 다른 자료형도 혼합해서 저장할 수 있습니다.
핵심 문장: 리스트는 순서가 있고, 변경 가능한 자료형입니다.

2) 리스트 생성 방법

# 숫자 리스트
numbers = [1, 2, 3, 4, 5]

# 문자열 리스트
fruits = ["apple", "banana", "cherry"]

# 자료형 혼합 리스트
mixed = [1, "apple", 3.14, True]
리스트는 정수, 문자열, 실수, 불리언 등 다양한 자료형을 함께 저장할 수 있습니다.

3) 리스트 출력(확인)

numbers = [1, 2, 3, 4, 5]
fruits = ["apple", "banana", "cherry"]
mixed = [1, "apple", 3.14, True]

print(numbers)
print(fruits)
print(mixed)

출력 결과

[1, 2, 3, 4, 5]
['apple', 'banana', 'cherry']
[1, 'apple', 3.14, True]

4) 인덱스(index)로 접근

fruits = ["apple", "banana", "cherry"]

print(fruits[0])  # apple
print(fruits[1])  # banana
print(fruits[-1]) # cherry (마지막)
👉
-1은 마지막 요소를 의미합니다.
인덱스로 요소를 “조회/수정”할 수 있습니다.

5) 리스트는 변경 가능합니다(mutable)

추가

numbers = [1, 2, 3]
numbers.append(4)      # 끝에 추가
print(numbers)         # [1, 2, 3, 4]

수정

numbers[0] = 99
print(numbers)         # [99, 2, 3, 4]

삭제

numbers.remove(2)      # 값 2 삭제(첫 번째 2)
del numbers[0]         # 인덱스로 삭제
print(numbers)
remove는 “값” 기준, del은 “인덱스” 기준으로 삭제합니다.

6) 리스트 특징 정리

저장 방식여러 값을 순서대로 저장
표현[ ] 대괄호
인덱스0부터 시작, 음수 인덱스 가능
변경가능 (mutable)
자료형 혼합가능

7) 리스트 vs 튜플 비교

구분리스트(list)
기호[ ]
변경가능 (mutable)
사용 예데이터가 추가/수정/삭제되는 경우
구분튜플(tuple)
기호( )
변경불가능 (immutable)
사용 예데이터가 변하면 안 되는 경우
✔ 데이터가 변해야 하면 리스트
✔ 데이터가 변하면 안 되면 튜플
