Python 딕셔너리(dict)

Key/Value 개념 · 생성/조회 · key 자료형 규칙(immutable)

1) 딕셔너리(dict)란?

👉
딕셔너리(dict)는 Key(키) : Value(값) 한 쌍으로 데이터를 저장합니다.
리스트처럼 “순서(index)”로 찾는 것이 아니라 키(key)로 바로 조회합니다.
  • { } 중괄호로 만들고 key: value 형태로 작성합니다.
  • mutable(변경 가능) → 추가/수정/삭제가 가능합니다.
  • 조회는 d[키] 또는 d.get(키)로 합니다.

기본 예제

d = {"name": "Alice", "age": 25}

print(d["name"])  # Alice
print(d["age"])   # 25
딕셔너리는 키로 바로 찾기 때문에 “검색 속도”가 빠른 편입니다.

2) 조회 방식: d[키] vs d.get(키)

둘 다 값을 꺼내지만, “키가 없을 때” 동작이 다릅니다.
d = {"name": "Alice"}

print(d["name"])      # Alice
print(d.get("name"))  # Alice

# 없는 키 조회
# print(d["age"])      # KeyError 발생
print(d.get("age"))   # None (기본값)
d[키] : 키가 없으면 KeyError가 발생합니다.
d.get(키) : 키가 없으면 기본으로 None을 반환합니다.

실전에서는 “키가 없을 수도 있는 상황”이면 get()이 안전합니다.

3) 딕셔너리 key 규칙(가장 중요)

핵심
딕셔너리의 key는 변경 불가능한(immutable) 자료형만 사용 해야 합니다.
키로 사용 가능 int, float, str, bool, tuple(단, 내부까지 immutable) 등
키로 사용 불가 list, dict, set 등 (mutable)

4) 키로 사용할 수 있는 자료형 예제

아래는 모두 정상 동작합니다.

정수(int) 키

d = {1: "one", 2: "two"}
print(d[1])  # one

문자열(str) 키

d = {"name": "Alice", "age": 25}
print(d["age"])  # 25

불리언(bool) 키

d = {True: "yes", False: "no"}
print(d[True])   # yes

튜플(tuple) 키

d = {(1, 2): "point"}
print(d[(1, 2)])  # point
튜플은 immutable이라 보통 키로 사용 가능합니다.
단, “튜플 내부”에 list 같은 mutable이 섞이면 키로 사용할 수 없습니다.

5) 키로 사용할 수 없는 자료형 예제

아래는 실행하면 TypeError(unhashable)가 발생합니다.

list 키 

# d = {[1, 2]: "value"}
# TypeError: unhashable type: 'list'
list는 mutable이라 키로 사용할 수 없습니다.

dict 키 

# d = {{1: 2}: "value"}
# TypeError: unhashable type: 'dict'
dict도 mutable이라 키로 사용할 수 없습니다.

set 키 

# d = {{1, 2}: "value"}
# TypeError: unhashable type: 'set'
set도 mutable이라 키로 사용할 수 없습니다.

튜플 내부에 list 포함 

# d = {(1, [2, 3]): "x"}
# TypeError: unhashable type: 'list'
키로 쓰려면 튜플 내부까지 전부 immutable이어야 합니다.

6) 딕셔너리에서 자주 하는 작업(추가/수정/삭제)

d = {"name": "Alice", "age": 25}

# 추가
d["city"] = "Seoul"

# 수정
d["age"] = 26

# 삭제
del d["city"]

print(d)  # {'name': 'Alice', 'age': 26}
딕셔너리는 mutable이라 key를 이용해 값 추가/수정/삭제가 가능합니다.
Tip: keys(), values(), items() 
d = {"name": "Alice", "age": 25}

print(d.keys())    # dict_keys(['name', 'age'])
print(d.values())  # dict_values(['Alice', 25])
print(d.items())   # dict_items([('name', 'Alice'), ('age', 25)])

정리

  • 딕셔너리는 key: value로 저장하며 키로 조회합니다.
  • 딕셔너리는 mutable이라 데이터 추가/수정/삭제가 가능합니다.
  • 튜플은 키로 가능하지만, 튜플 내부에 mutable이 있으면 불가능합니다.
