파일 입출력(File I/O) - 열기 / 쓰기 / 읽기

open mode encoding with read write

1. 파일 입출력의 개념

👉
파일 입출력(File I/O)은 파일에 데이터를 저장(쓰기)하거나 파일에서 데이터를 가져오는(읽기) 작업입니다.
기본 흐름은 파일 열기 → 읽기/쓰기 → 파일 닫기 입니다.

왜 파일을 사용하나요?

  • 프로그램 종료 후에도 데이터를 저장해 둘 수 있습니다.
  • 텍스트(로그), 설정, 결과 등을 파일로 남겨 관리할 수 있습니다.
  • 다른 프로그램과 데이터를 교환할 수 있습니다.

기본 규칙

  • 텍스트 파일은 보통 utf-8 인코딩을 사용합니다.
  • 읽기("r")는 파일이 없으면 에러가 발생합니다.
  • 쓰기("w")는 기존 내용이 삭제될 수 있으니 주의합니다.

2. open() 함수와 모드(mode)

2-1) open() 기본 형태

텍스트 파일을 다룰 때 가장 많이 쓰는 형태입니다.

f = open("sample.txt", "r", encoding="utf-8")
# ... 읽기/쓰기 작업 ...
f.close()
중요
파일 작업이 끝나면 반드시 close()로 닫아야 합니다. 닫지 않으면 저장이 누락되거나 파일이 잠길 수 있습니다.

2-2) 모드(mode) 정리

모드 의미 특징
"r" 읽기 파일이 없으면 에러가 발생합니다.
"w" 쓰기 기존 내용이 삭제되고 새로 저장됩니다. 파일이 없으면 생성됩니다.
"a" 추가 쓰기 기존 내용 뒤에 이어서 저장됩니다. 파일이 없으면 생성됩니다.
"r+" 읽기 + 쓰기 읽고 쓰기가 모두 가능합니다. 파일이 없으면 에러가 발생합니다.
👉
로그처럼 계속 누적 저장이 필요하면 a 모드를 사용합니다.
w 모드는 실행할 때마다 파일이 초기화될 수 있습니다.

3. with open() (권장 방식)

3-1) with를 사용하는 이유

  • 자동으로 close()가 호출되어 안전합니다.
  • 실수로 파일을 닫지 않는 문제를 줄여줍니다.

예제

with open("sample.txt", "r", encoding="utf-8") as f:
    data = f.read()

print(data)

4. 파일 쓰기(write) 예제

4-1) 새로 쓰기 (w)

파일이 없으면 생성되고, 있으면 기존 내용이 삭제됩니다.

with open("sample.txt", "w", encoding="utf-8") as f:
    f.write("안녕하세요\n")
    f.write("파이썬 파일 입출력입니다.\n")
    f.write("w 모드는 기존 내용을 삭제하고 새로 저장합니다.\n")

4-2) 이어 쓰기 (a)

파일 끝에 내용을 추가합니다.

with open("sample.txt", "a", encoding="utf-8") as f:
    f.write("이 줄은 a 모드로 추가된 내용입니다.\n")
주의
write()는 문자열만 저장합니다. 숫자는 str()로 변환해서 저장합니다.

5. 파일 읽기(read) 예제

5-1) 전체 읽기 (read)

with open("sample.txt", "r", encoding="utf-8") as f:
    data = f.read()

print(data)

5-2) 한 줄 읽기 (readline)

한 줄씩 읽습니다. 줄 끝의 \n이 포함될 수 있습니다.

with open("sample.txt", "r", encoding="utf-8") as f:
    line1 = f.readline()
    line2 = f.readline()

print("1:", line1)
print("2:", line2)

5-3) 여러 줄 읽기 (readlines)

각 줄을 원소로 가지는 리스트로 읽습니다.

with open("sample.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()

print(lines)  # ["...\\n", "...\\n", ...]

6. 숫자 데이터 저장 후 다시 숫자로 읽기

파일에서 읽은 값은 문자열이므로 정수/실수로 변환해야 합니다.

# 1) 저장 (숫자 -> 문자열)
scores = [90, 80, 70]
with open("scores.txt", "w", encoding="utf-8") as f:
    for s in scores:
        f.write(str(s) + "\n")

# 2) 읽기 (문자열 -> 정수)
with open("scores.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()

scores2 = list(map(int, lines))
print(scores2)  # [90, 80, 70]
👉
숫자를 저장할 때는 str()로 변환합니다.
숫자를 읽을 때는 int() 또는 float()로 변환합니다.

7. 파일이 없을 때 예외 처리

try:
    with open("no_file.txt", "r", encoding="utf-8") as f:
        print(f.read())
except FileNotFoundError:
    print("파일이 존재하지 않습니다.")
체크 포인트
- "w"는 기존 내용이 삭제됩니다.
- 파일에서 읽은 값은 문자열이므로 숫자는 int()로 변환합니다.
- with open()을 사용하면 자동으로 닫힙니다.
